# Dgen CLI v1.0.0

Dgen Cli is a npm package which allows us to generate base projects for our discord bot
## How to Install?

Use the package manager [npm](https://npmjs.com/) to install Dgen.

```bash
npm i -g dgen-cli
```

## How to use ?
### Type `npx dgen` or `dgen` after installing module
```bash 
npx dgen or dgen
```
